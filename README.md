- `react-hooks`
- `redux`
- `redux-persist`
- `redux middleware` \[**Thunk & logger**\]

## INSTALLATION

1. `npm install` or `yarn`
2. `npm start` or `yarn start` for run the project
3. `yarn test` for run the test
